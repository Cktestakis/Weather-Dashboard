# Weather-Dashboard
This will give you accessibility to see the weather outlook for multiple cities.

## Technologies used
This project was created with the following technologies:
* HTML
* CSS
* JavaScript
* Jquery
* Moment.js
* Open Weather API
* Bootstrap
* Font Awesome
